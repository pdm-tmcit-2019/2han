# WerewolfSharpGUI
Werewolf world の Json ファイルデータをGUIで表示するためのツールです。

# 更新状況 (Develop-ver1.5.0)
C++を呼び出す機能を実装しました。まだ整数型のみですが</br>

## 実装されている機能

- 会話ログの表示 💬
- 会話の送信（データベースへの保存のみ）👨
- 生存者数の数え上げと表示 👬
- そのフェーズで分かっている役職ボードの表示 🐺
- Jsonldファイルの書き出し 📝
- C++で作成した関数の呼び出し（整数型のみ）📞<font color="Red"> ← New!!</font>

"C++呼び出し"ボタンが追加されました。</br>
